## SeSAC Mini Project1 

## 📌Overview
이 프로젝트는 서울시 부동산 실거래가 정보와 공공 데이터를 활용하여, 사용자의 라이프스타일에 맞춘 서울의 최적의 거주지를 추천합니다.  
지도 기반 시각화를 제공하여, 직관적인 UI로 살기 좋은 동네를 확인할 수 있는 웹 서비스입니다. 

## 📌Period
2025.02.20 ~ 2025.02.26
